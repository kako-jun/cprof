# 概要

## 作ろうと思ったわけ

カラーマネジメントを扱う場面で、「このプロファイルの色域ってどれくらい広いんだろう」と思っても、
数値だけでは実感がわかない。sRGB と Adobe RGB の違いを「広色域です」と言葉で説明されても、
頭の中で 3D 空間を思い描くのは難しい。

だったら、実際に 3D で見せればいい。
それがこのツールを作り始めた動機だ。

## 何をするツールか

ICC カラープロファイルのファイルをブラウザにドロップすると、
そのプロファイルが表現できる色の範囲（色域）を 3D 空間に描き出す。

CIE XYZ 色空間のなかで、どの色が「そのプロファイルの範囲内」なのかを立体として表示する。
回転させたり、ズームしたり、断面を切ったりしながら、色域の形を直感的に把握できる。

複数のプロファイルを重ねて表示すれば、「sRGB に対して Display P3 はここが広い」という差が
一目でわかる。

## 誰のためのツール

- カラーマネジメントを学びはじめたデザイナーや写真家
- モニターや印刷機のプロファイルを比較したいプロフェッショナル
- 色空間の仕様を視覚的に確認したいエンジニア
- ただ「色の世界」を眺めて楽しみたい人

専門知識がなくても、ブラウザさえあればすぐに使える。
インストール不要、サーバー不要、完全にブラウザだけで動く。

## 扱えるプロファイル

ICC v2/v4 形式の `.icc` または `.icm` ファイルに対応している。
自分のモニターやプリンターのプロファイルを読み込むことも、
同梱のサンプルプロファイルで試すことも、どちらもできる。

同梱のサンプル:

| プロファイル | 位置づけ |
|---|---|
| sRGB | Web・一般ディスプレイの標準 |
| Adobe RGB | 印刷・写真編集向けの広色域 |
| Display P3 | Apple 製品・HDR コンテンツ向け |
| Rec.2020 | 4K/8K 映像向けの超広色域 |
| ProPhoto RGB | RAW 現像向けの最大級色域 |

## 色の世界を「見る」ということ

色空間の議論はしばしば数値とグラフの話になりがちだ。
しかし本来、色は感覚的なものだ。

このツールは、抽象的になりやすい色域の概念を、
実際に手で回して確かめられるかたちにする。
「広い」「狭い」を体感として持ってもらうことが、このツールの核心にある。
