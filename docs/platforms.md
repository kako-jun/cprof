# 動作環境

## ブラウザ対応

| ブラウザ | 対応バージョン | 備考 |
|---|---|---|
| Chrome | 90 以降 | 推奨。動作確認済み |
| Firefox | 88 以降 | 動作確認済み |
| Safari | 14 以降 | macOS・iOS で動作確認済み |
| Edge | 90 以降 | Chromium ベース版。動作確認済み |

古いバージョンのブラウザでは動作しない場合がある。
ブラウザを最新版に更新することを推奨する。

---

## WebGL 要件

3D 色域の表示には WebGL が必要だ。
現行の PC・スマートフォンのほとんどは対応している。

### WebGL が使えない環境

以下の場合は 3D 表示が動作しない:

- ブラウザの設定でハードウェアアクセラレーションが無効になっている
- 非常に古いグラフィックドライバーを使用している
- 一部の仮想環境やリモートデスクトップ環境

WebGL が使えない場合でも、2D 色度図やカバレッジダッシュボードは表示される。

### 確認方法

ブラウザで `chrome://gpu`（Chrome の場合）を開くと WebGL の状態を確認できる。
「WebGL: Hardware accelerated」と表示されていれば問題ない。

## モバイル対応

スマートフォン・タブレットのブラウザでも動作する。
ただし 3D 操作はタッチジェスチャーで行う:

- 1 本指スワイプ: 視点を回転
- ピンチイン・アウト: ズーム
- 2 本指スワイプ: 視点を平行移動

画面が小さい場合、レイアウトが縦並びに切り替わる。
プロフェッショナルな用途では PC での使用を推奨する。

## 対応ファイル形式

### ICC プロファイル

| 拡張子 | 内容 |
|---|---|
| `.icc` | ICC プロファイル（標準的な拡張子） |
| `.icm` | ICC プロファイル（Windows でよく使われる拡張子） |

### ICC バージョン

| バージョン | 対応状況 |
|---|---|
| ICC v2 | 対応 |
| ICC v4 | 対応 |

### 対応する色空間タイプ

| タイプ | 対応状況 |
|---|---|
| RGB 色空間 | 対応 |
| グレースケール | 対応 |
| CMYK | 現時点では非対応（ロードマップに記載） |
| Lab | 現時点では非対応 |

## 同梱サンプルプロファイル

アプリには以下のプロファイルが同梱されており、ファイルなしで即座に試せる:

| プロファイル名 | バージョン | 特徴 |
|---|---|---|
| sRGB | v2 / v4 | Web 標準。最も広く普及している |
| Adobe RGB | v4 互換 | 印刷・写真編集で使われる広色域 |
| Display P3 | v4 | Apple 製品や DCI-P3 対応モニター向け |
| Rec.2020 | v4 | 4K/8K 映像・HDR コンテンツ向けの超広色域 |
| ProPhoto RGB | v4 | RAW 現像など最大級の色域 |

---

## ネットワーク要件

基本的な使用には特別なネットワーク要件はない。
初回アクセス時にページのアセット（JavaScript、Three.js など）を読み込む必要があるが、
その後はオフラインでも動作する（ページをリロードしない限り）。

共有リンクを使う場合は、リンクを開くために一度オンラインである必要がある。

---

## プライバシー

アップロードしたプロファイルはサーバーに送信されない。
すべての処理はブラウザ内で完結している。
手元のプロファイルをクラウドに上げる必要はなく、安心して使える。
